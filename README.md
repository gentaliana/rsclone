## :gear: Setup

```bash
# Clone repo
$ git clone https://github.com/gentaliana/rsclone

$ cd rsclone

# Switch to the develop branch
$ git checkout develop

# Install all dependencies
$ npm install

# Run
$ npm run start

# Go to http://localhost:3000/ on your web browser.
```
